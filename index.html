Essential Excel Formulas

Master the most important formulas with examples and explanations.

Basic Math Functions

SUM

=SUM(A1:A10)

Adds all numbers in a range

AVERAGE

=AVERAGE(A1:A10)

Calculates the mean of numbers

COUNT

=COUNT(A1:A10)

Counts cells with numbers

Text Functions

LEFT

=LEFT(text, num_chars)

Extracts characters from left

RIGHT

=RIGHT(text, num_chars)

Extracts characters from right

CONCATENATE

=CONCATENATE(text1, text2)

Joins text together

Lookup Functions

VLOOKUP

=VLOOKUP(lookup_value, table_array, col_index, [range_lookup])

Vertical lookup in table

XLOOKUP

=XLOOKUP(lookup_value, lookup_array, return_array)

Modern lookup function

Logical Functions

IF

=IF(logical_test, value_if_true, value_if_false)

Conditional logic

IFS

=IFS(logical_test1, value1, logical_test2, value2)

Multiple conditions

Excel Tips & Tricks

Time-saving shortcuts and techniques to boost your productivity.

Navigation Shortcuts

Ctrl + Arrow Jump to edge of data
Ctrl + Home Go to cell A1
F5 Open Go To dialog
Ctrl + Page Up/Down Switch between sheets

Selection Tricks

Ctrl + A Select all data in region
Shift + Arrow Extend selection
Ctrl + Shift + Arrow Select to edge of data
Double-click border Select to edge

Editing Shortcuts

F2 Edit current cell
Ctrl + Enter Fill selected cells
Alt + Enter New line in cell
Ctrl + D Copy from cell above

Formula Tricks

F4 Cycle absolute references
Ctrl + ` Toggle formula view
F9 Evaluate formula parts
Ctrl + Shift + Enter Array formulas (legacy)

Begin with the Beginner Essentials track, then move to Formulas & Logic. Use the filters to find beginner-friendly resources.
